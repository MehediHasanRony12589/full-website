Advance Home Automation (IoT)

Authors: Mehedi Hasan Roni, Md Tasrik Hossain, Md Siyam Sorkar
Institution: Cumilla Polytechnic Institute
Organization: Creativity Icon

Abstract

With the rapid evolution of smart technologies, automation has transformed the way modern homes operate. This thesis introduces Advance Home Automation, a budget-friendly, compact, and highly adaptable automation system designed to control essential household devices such as lights, fans, water pumps, regulators, lifts, and generators...

Keywords: Home Automation, IoT, NFC, Smart Home, Wi-Fi Control, Bluetooth Connectivity, Remote Access, Smart Energy, Affordable Smart Systems

1. Introduction

In an era driven by automation and connectivity, smart homes are no longer a futuristic luxury but a modern necessity. However, most advanced home automation systems remain costly, complex, or inaccessible to a large segment of society, especially in developing countries. Recognizing this gap, we developed Advance Home Automation, a locally designed and implemented system in Bangladesh, offering affordable and accessible smart control over key electrical appliances. This system enables automatic control of home electronics via alarms and schedules, offering both Wi-Fi and offline Bluetooth access, alongside NFC-triggered night mode and support for universal remote controllers. The device is currently installed in multiple residential setups and has demonstrated excellent performance in daily use.

2. Problem Statement

Despite the availability of smart home technologies, most systems are:

  • Expensive
  • Dependent on continuous internet
  • Limited in control range
  • Difficult to install or integrate

3. Objectives

  • To develop a cost-effective and compact smart controller.
  • To ensure remote and offline control (via Wi-Fi and Bluetooth).
  • To automate essential household utilities: light, fan, water pump, lift, regulator, generator.
  • To support custom alarms and schedules for automation.
  • To provide an NFC-based night mode activation.
  • To allow control via TV/AC remotes, increasing user flexibility.
  • To integrate with a custom mobile application (under development).

4. Literature Review

SmartThings offer extensive automation capabilities. However, these systems:

  • Are priced at a premium.
  • Require cloud connectivity.
  • Often do not support offline operation.
  • May not be compatible with local infrastructure in Bangladesh.

    • Studies on IoT-based smart homes ([1], [2]) emphasize the importance of local control, data privacy, and cost-effective implementation. Our work builds upon these principles, providing a practical alternative specifically designed for Bangladeshi users.

5. Methodology

The Advance Home Automation system was developed in multiple phases:

Phase 1: Design & Planning

  • Identified key appliances to automate.
  • Chose components for low-cost, low-power operation.

Phase 2: Hardware Development

  • Used microcontrollers (e.g., ESP8266 for Wi-Fi and Bluetooth capability).
  • Integrated relay modules for switching high-voltage appliances.
  • Built custom PCB for compactness.

Phase 3: Software & Scheduling

  • Developed embedded code for automation via alarms and timers.
  • Configured NFC triggers using Android-compatible NFC tags.
  • Enabled cross-platform access using standard remote controllers.

Phase 4: Testing & Installation

  • Installed prototypes in multiple residential buildings.
  • Evaluated functionality under different scenarios (with and without internet).

6. System Design and Implementation

The Advance Home Automation system is built on a modular architecture that combines affordability, scalability, and reliability. The hardware and software components are integrated to deliver real-time, scheduled, and remote-controlled automation capabilities.

6.1. Hardware Components

The core hardware module consists of the following components:

  • Microcontroller: ESP8266 (NodeMCU) — provides Wi-Fi and Bluetooth capabilities.
  • Relay Modules: Two-channel relays for controlling high-voltage AC appliances such as lights, fans, water pumps, and generators.
  • Power Supply Circuit: Includes voltage regulation to provide stable 5V/3.3V to microcontroller and relays.
  • NFC Module: Allows users to switch to “Night Mode” by simply tapping an NFC tag.
  • Infrared Receiver: Compatible with standard TV and AC remotes for manual triggering.
  • Printed Circuit Board (PCB): Custom-designed for compact integration and durability.

6.2. Software Architecture

  • Embedded C/C++ was used to program the microcontroller.
  • Alarm and scheduling logic was implemented using Real-Time Clock (RTC) libraries.
  • Wi-Fi connectivity was configured through MQTT and HTTP protocols for remote commands.
  • Bluetooth fallback enabled direct control in offline scenarios via a dedicated mobile app (in development).
  • NFC tags were programmed with predefined actions to activate night mode (e.g., dimming lights, disabling fans, etc.).

6.3. Control Interfaces

  • Remote Control: Universal IR remote compatibility.
  • Mobile App (Upcoming): A cross-platform app will provide an intuitive GUI for managing all connected devices, schedules, and modes.
  • Voice Assistants (Planned): Future integration with voice-based systems like Google Assistant or Alexa.

6.4. Installation

  • Devices are plug-and-play compatible with traditional switchboards and home wiring.
  • Already installed in flats, homes, and apartments across several locations in Bangladesh.

7.1. User Feedback

  • Real‑world Deployment: Installed in diverse residential settings across Bangladesh.
  • Affordability: Approximately 80 % cheaper than comparable imported systems.
  • Ease of Use: No technical expertise required.
  • Compact Design: Fits behind traditional switchboards.
  • Flexibility: Seamless operation both online and offline.

7.2. Sustainability

Scheduled shutdown of lights and fans has reduced unnecessary energy consumption, promoting efficient electricity use in urban areas prone to energy waste.

8. Limitations

  • No Mobile App Yet: Full user interface is still under development, limiting centralized control.
  • Limited Device Support per Unit: The current module supports only 2–4 devices, requiring expansion for larger setups.
  • Manual Configuration: Initial installation and setup require basic electrical knowledge, which may limit accessibility for all users.
  • Lack of Security Layer: Currently, the system lacks encryption for remote commands, posing potential risks in connected environments.

9. Future Work

  • Mobile App Launch: Development and release of a fully-featured mobile app for automation, scheduling, and remote access.
  • Device Expansion: Integration of new modules to support air conditioners, smart locks, and security cameras.
  • Voice Assistant Integration: Compatibility with Google Assistant and Amazon Alexa for voice-based control.
  • Security Enhancements: Implementation of end-to-end encrypted communication to protect user data and commands.
  • Cloud Analytics: Real-time energy usage and appliance monitoring via cloud dashboards.
  • AI-Driven Automation: Incorporating AI to learn user behavior and dynamically optimize schedules and energy usage.

10. Conclusion

This thesis presents Advance Home Automation, a practical, affordable, and efficient system designed to meet the automation needs of everyday homes in Bangladesh. By integrating Wi-Fi, Bluetooth, NFC, and universal remote features, this system addresses a wide range of user scenarios — from internet-free environments to full remote operation. Most importantly, the system is already deployed and successfully functioning in several homes, flats, and apartments in Bangladesh, confirming its real-world viability and demand. Its compact size, low cost, and ease of installation make it ideal for both individual households and developers of modern smart apartments. With ongoing improvements like mobile application development and expanded device support, this innovation has the potential to become a standard solution in Bangladesh’s transition toward smart living.
